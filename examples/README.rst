Examples
========

This directory contains example scripts demonstrating the capabilities of LoopResources.

The examples are automatically converted to documentation using Sphinx Gallery.
Each Python script that starts with ``plot_`` will be executed and its output
included in the documentation.

Example naming convention:
- ``plot_basic_usage.py`` - Basic usage examples
- ``plot_drillhole_database.py`` - DrillholeDatabase examples
- ``plot_advanced_filtering.py`` - Advanced filtering examples
